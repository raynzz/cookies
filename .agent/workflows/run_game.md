---
description: How to run the spaceship racer game locally
---

1. Ensure Node.js is installed.
2. Navigate to the project directory:
   ```bash
   cd /Users/raynerteran/.gemini/antigravity/scratch/spaceship-racer
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open the local URL (e.g., http://localhost:5173) in your browser.
