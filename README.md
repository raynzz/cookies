# Spaceship Racer

A minimalist spaceship racing game with flow and wave simulations.

## How to Run

Since the environment had issues running `npm` directly, please follow these steps to run the game on your machine:

1.  Open a terminal in this directory:
    ```bash
    cd /Users/raynerteran/.gemini/antigravity/scratch/spaceship-racer
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the URL shown in the terminal (usually `http://localhost:5173`).

## Controls

-   **Arrow Keys** or **WASD**: Move the spaceship.
-   **Goal**: Survive the flow and progress through 5 levels of increasing intensity.

## Features

-   **5 Unique Levels**: From "The Calm" to "Singularity".
-   **Wave Simulation**: Procedural background waves that react to speed.
-   **Minimalist Aesthetics**: Neon visuals on a dark background.
